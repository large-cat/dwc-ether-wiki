<h level="2">DMA Controller Overview</h>
<info>
  <p>Chapter: [[架构]]</p>

  <p>Confidence: <code>high</code></p>

<p>The DMA has independent Transmit (Tx) and Receive (Rx) engines, and a CSR space. The Tx engine transfers data from system memory to the device port (MTL), whereas the Rx engine transfers data from the device port to system memory. The DMA engine uses descriptors to efficiently move data with minimal application CPU intervention. The DMA supports up to 8 Tx and 8 Rx Descriptor lists (or DMA channels). The descriptor lists are forward-linked rings. Each descriptor can point to a maximum of two buffers in system memory.</p>
